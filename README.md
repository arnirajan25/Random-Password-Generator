# Random Password Generator 
A fast, lightweight, and cryptographically secure command-line password generator written in Go. Easily generate strong, customizable passwords to protect your digital accounts right from your terminal.

---

## Features

* Secure Randomness: Uses Go's `crypto/rand` package to generate passwords using cryptographically secure values rather than predictable pseudo-random seeds.
* Customizable Length: Tailor your password length dynamically based on your security policies.
* Zero Dependencies: Built entirely using Go's native standard library—no bulky node modules or third-party packages required.
* Highly Portable: Can be compiled into a single executable binary that runs anywhere.

---
